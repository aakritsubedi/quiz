# quiz-web 
- VueJs Application

## Project Setup 
- clone the repository

```bash
$ git clone git@github.com:aakritsubedi/quiz-web.git
```

## To install all the required packages

```bash
$ npm install
```
OR

```bash
$ yarn
```

## To run the server

```bash
$ npm serve
```
OR

```bash
$ yarn serve
```

## App will be running at: Local: http://localhost:8080/
